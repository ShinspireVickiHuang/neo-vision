## 重點規則（摘要）
- 表單名稱必選：未選表單時結果固定為 0 筆。
- 頁面初始顯示 0 筆（表格隱藏）。
- 任一查詢條件變更後，結果列表即時更新。
- 查看對象支援「案場 / 組織」兩種模式（透過彈窗選取）；預設為「組織 + 自己」。
- 處理種類只控制「送單人」欄位顯示，不直接影響篩選；且只在「組織 + 自己」時顯示。
- 結果依狀態切換 Tabs：審批中 / 被取消 / 被暫停 / 已結案（已結案最後且不顯示筆數）。
- 0 筆結果時隱藏表格、仍顯示筆數。
- 表格欄位會依表單類型切換（考核表/會議記錄/其他）；查看對象為組織時會多一欄「組織」。
- 查詢區提供「申請」按鈕（Prototype 入口）。

## 查詢條件（欄位與互動）

### 1) 表單名稱（必選）
- 下拉選單，選項由假資料 `formName` 去重產生。
- 未選表單：結果固定為 0 筆。

### 2) 表單單號
- 文字輸入，包含比對（contains）。

### 3) 查看對象（案場 / 組織）
- 組成：下拉（案場/組織）＋只讀輸入框＋「選擇」按鈕。
- 案場：彈窗清單提供「不限」與各案場。
	- 選「不限」：不限制案場（可查該表單全部資料）。
- 組織：彈窗顯示樹狀結構（部門/人員），並提供「選擇自己」。

### 4) 處理種類 / 送單人（條件顯示）
- 處理種類顯示條件：僅在「查看對象 = 組織」且選取為「自己」時顯示。
- 選「本人承辦表單」時顯示「送單人」欄位；其他選項則隱藏並清空。
- 送單人為包含比對（contains）。

### 5) 申請日期（起迄）
- 任一日期變更即觸發查詢。

## 結果區（狀態 Tabs + 表格）

### 狀態 Tabs
- 順序：審批中 / 被取消 / 被暫停 / 已結案。
- 顯示筆數：除已結案外，其餘 Tab 顯示（N）。
- 切換 Tab：更新表格與筆數。

### 表格欄位
- 依表單類型切換：
	- 人事｜試用員工考核表：案場 /（組織）/ 員工姓名 / 現職單位 / 狀態 / 送單人 / 申請日期 / 目前關卡 / 操作
	- 會議記錄：週會/月會 /（組織）/ 時間 / 地點 / 主席 / 紀錄 / 狀態 / 送單人 / 申請日期 / 目前關卡 / 操作
	- 其他：案場 /（組織）/ 申請人 / 狀態 / 送單人 / 申請日期 / 目前關卡 / 操作

## 匯出
- 匯出範圍：匯出「目前狀態 Tab」下的查詢結果。
- 匯出格式：下載 CSV。
- 匯出欄位：依表單類型輸出核心欄位，且不包含：狀態 / 送單人 / 申請日期 / 目前關卡 / 操作。
